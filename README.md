# Jeevita Bhupati — Portfolio Website

A personal portfolio website with a Harry Potter theme, built using HTML and CSS.

---

## Website Structure

The site is a single `index.html` file. Here's how it flows from top to bottom:

### 1. Navigation Bar
A fixed top nav with links to: About, Projects, Hackathons, Certificates, Skills, and Contact.

### 2. Intro (Scroll Animation)
A full screen black page. As you scroll, a profile photo and name fade in and scale up, then fade out built using CSS.

### 3. About — *The Marauder's Notes*
A two column section with academic details on the left (college, branch, CGPA, clubs) and a personal bio on the right.

### 4. Interactive Map
A parchment map image with a button — *"I solemnly swear that I am up to no good"*. Clicking it reveals a short fun bio, using a pure CSS checkbox trick.

### 5. Projects — *Things I've Conjured*
A horizontally scrollable row of project cards, each with a title, description, and link.

### 6. Hackathons — *Battles I've Fought*
A vertical list of hackathon participations with event name, date, and result.

### 7. Certificates — *Scrolls of Achievement*
A horizontally scrollable row of certificate images that lift up on hover.

### 8. Skills — *Spells in Progress*
Pill style tags for technical skills.

### 9. Soft Skills — *Not Even on O.W.L.s or N.E.W.T.s*
Pill style tags for soft skills.

### 10. Languages — *My Coding Grimoire* & *Parseltongue & Beyond*
Two tag groups — one for programming languages, one for spoken languages.

### 11. Contact — *Contact Me the Muggle Way*
Links to email, LinkedIn, GitHub, and phone number.
